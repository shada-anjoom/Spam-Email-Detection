# 📧 Spam Email Detection

## Overview
This project is a Machine Learning application that detects whether an email is **Spam** or **Not Spam** using Natural Language Processing (NLP).

## Features
- Email preprocessing and cleaning
- TF-IDF Vectorization
- Multinomial Naive Bayes classifier
- Model training and evaluation
- Confusion Matrix
- ROC Curve
- Classification Report

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Model Performance
- Accuracy: **97.73%**
- Precision: **98%**
- Recall: **93%**
- F1-Score: **95%**
- ROC-AUC: **0.9979**
